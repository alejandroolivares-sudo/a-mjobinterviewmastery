# A&M English - Job Interview Mastery Web Platform This is an interactive web-based training platform designed to help students master job interviews conducted in English[cite: 2]. Built entirely using modern, responsive frontend frameworks, this site guides candidates away from rigid script memorization and focuses on developing natural fluency and professional rapport[cite: 2, 3]. ## 🚀 Key Features * **Interactive Main Guide:** Outlines the 19 most common interview questions using a strict 3-step hierarchy: The Question, The Explanation, and High-Quality Answer Examples[cite: 2]. * **The STAR Method Framework:** Integrates embedded multimedia guidance detailing how to properly structure situational responses (Situation, Task, Action, Result)[cite: 2]. * **Essential Phrase Bank:** Quick-access linguistic patterns to improve structural delivery, vocabulary choices, and professional phrasing[cite: 2]. * **Interactive Practice Simulator:** Built-in audio recording feature via the browser's MediaDevices API allowing students to speak, record, and playback their practice answers seamlessly. * **Simulated HR Reviewer Feedback:** An evaluation rubric reflecting real HR recruiter standards—grading the user on Pronunciation, Vocabulary, Fluency, Grammar, and Phrasal alignment[cite: 2]. * **Actionable Pre-Interview Checklists:** Built-in career coaching modules teaching company research techniques and the physiological "Mirror Technique" for high-pressure scenarios[cite: 2, 3]. ## 🛠️ Technology Stack * **HTML5** (Semantic page structure) * **CSS3** (Custom responsive grid design & corporate theme parameters) * **Vanilla JavaScript** (Tab management and native browser Web Audio API implementation) ## ⚠️ Hosting Requirements To ensure the **Audio Recording Feature** works correctly, this site **must be hosted over HTTPS**. Modern web browsers strictly block microphone access on insecure (HTTP) origins. Free automated HTTPS deployments are fully supported via GitHub Pages, Netlify, or Vercel.
